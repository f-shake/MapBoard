# MapBoard 地图画板

一个以地图为底图的画图板，可以在其上绘制各种图形，包括线、面、点等，设置不同的样式，设置不同的标签
A map-based drawing board on which you can draw various shapes, including lines, faces, points, etc., set different styles, and set different labels.
