# 地图画板

## 简介

《地图画板》是一个以地图为底图的矢量画板软件，支持图层管理、分类上色、标签显示、简单分析、快速编辑等功能，上手简单、运行速度快。同时，配有《瓦片地图下载器》和《GPX工具箱》两个辅助软件，可以用于缓存离线地图、处理GPX文件。
